# ChatConnect

A real-time chat application built using React, Node.js, Express, and Socket.io.

## Live Demo

https://chat-app-connect-rt.vercel.app

## Demo

https://drive.google.com/drive/folders/1zllKTMNrUVUC6vsqbU8TR_GCg-rvLKoK?usp=sharing

## Features

- Real-time messaging
- Username-based login
- Message timestamps
- Connection status
- Previous messages on join
- Responsive UI
- Logout functionality

## Tech Stack

Frontend
- React
- Vite
- React Router
- Socket.io Client

Backend
- Node.js
- Express
- Socket.io

Deployment
- Vercel
- Render
